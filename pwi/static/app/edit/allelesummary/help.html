Allele Summary functionality:

  • Wild type alleles are included.
  • Sort order: (1)Transmission Status descending (so "Cell Line" comes last), (2)Allele Status, (3)Symbol in ASCII order.
  • Only the first 1,000 matching records are displayed. To get all results if more than 1,000 records match, click the "View All Matches" link at the top of each table, or download the tab-delimited file.
  • The MP Annotations column has 3 possible values:
    • blank - there are no annotations.
    • has data - there is at least 1 MP annotation to an "abnormal" term.
    • no abnormal phenotype observed - all annotations have the "norm" qualifier.
  • The Disease Annotations column lists ALL disease terms annotated to any genotype involving the allele, including "NOT" annotations.
Allele Summary for a Marker functionality:
  • The URL supports /pwi/edit/allelesummary?marker_id=MGI:(number)
  • This summary returns all alleles for a marker with any allele status value. It does not include alleles using "involves" or "expresses" relationships.
Allele Summary for a Reference functionality:
  • The URL supports /pwi/edit/allelesummary?refs_id=J:(number)
  • This summary includes only Alleles with a direct association to the Reference. This includes relationship types: Indexed, Original, Molecular, Transmission, Expression-only, Used-FC, and Not Used.
