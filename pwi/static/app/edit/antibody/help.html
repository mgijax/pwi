P-WI Antibody Editor Interface functionality

General Description

The Antibody module is a display and edit page all in one. A user must be logged in to edit but can search without logging in.

Keyboard Short-cuts

  • Add = Ctrl-Alt-A
  • Delete = Ctrl-Alt-D
  • Search = Ctrl-Alt-S
  • Clear = Ctrl-Alt-C
  • Modify = Ctrl-Alt-M
  • First = Ctrl-Alt-F
  • Previous = Ctrl-Alt-P
  • Next = Ctrl-Alt-N
  • Last = Ctrl-Alt-L

Color-coded Labels

  • Thistle = search only
  • Green = edit
  • Blue = read only
  • Yellow = current row

Supported Browsers

  • Chrome and FireFox are supported on both PC and Mac computers.
  • Safari is NOT supported for the PWI. There are features that do not work.

Search by Dates

  • Valid date formats for searching are:
    • mm/dd/yyyy or mm-dd-yyyy
    • mm/dd/yy or mm-dd-yy
    • yyyy/mm/dd or yyyy-mm-dd
  • Searching for a range of dates:
    • equal to: enter a single date
    • greater than: > mm/dd/yyyy
    • greater than or equal to: >= mm/dd/yyyy
    • less than: < mm/dd/yyyy
    • less than or equal to: <= mm/dd/yyyy
    • between two dates (inclusive): mm/dd/yyyy..mm/dd/yyyy

Searching

  • All fields may be used for searching.
  • They may be searched individually or in combination. In some, but not all, cases when more than one field is used in a search, it is treated as an AND search.
  • A search of the Name field will simultaneously search Names and Aliases. A search of the Alias field will only search Aliases.
  • A search with no parameters returns all antibodies.

Wildcards

  • Wildcards (%) may be used for most fields.
  • A wildcard may be used at the beginning, middle and/or end of your search test: e.g. %paired%, casein% or AK32%43%.

Creating New Antibodies

  • The fields required to create an antibody are: Name, Antigen ID, and Primary Reference. All other fields are optional.
  • Each antibody must have one primary reference; all other references should be type = related.
  • Aliases are not required to have a J#; they may be entered without.
  • Not all antibodies have an associated gene. These antibodies recognize non-mouse genes, e.g. B-galactosidase and GFP, and are used for in situ reporter assays (knock-in and transgenic) and recombinase reporter assays.
  • The ‘Companies (for Notes)’ field is used to create a note for a commercially available antibody. Start typing a company name, e.g. Santa. The field autofills from a Simple Vocabulary (called 'GXD Antibody Companies'), pick the company, and tab out of the field. A note, 'Antibody obtained from Santa Cruz Biotechnology,' will appear in the notes field.
  • The ‘Companies (for Searching)’ field is used to minimize typing when searching for existing antibodies. Start typing a company name, pick the company from the autofill, and tab out of the field. The company name surrounded by wildcards (%) will appear in the notes field.
  • New antibodies can be created by “copying” existing antibodies. To do so, pull up the antibody whose values you want to use; deselect it from the search return; add a primary J#; hit Add.

Antigen Information

  • The Strain, Tissue and Cell Line are validated fields that have an autofill dropdown to help with entry. New items can be added to this list by clicking yes in the dialogue box when asked.
  • The Antigen Source defaults are the same as are used in the probes module.
    • If the organism is not mouse, laboratory or Not Specified, then the strain field default should be Not Applicable.
    • If a Tissue is entered, then the Cell Line field default should be Not Applicable. (Curators can override it, but the default is Not Applicable.)
    • If a Cell Line is entered, then the Age default should be Not Applicable. (If there is some cell line related age information curators wish to record, they should put it in the Tissue Description notes field.)
  • If you choose Organism = Other (see notes), you must enter explanatory text into the Notes field.
  • The Strain, Tissue, and Cell Line field use controlled vocabularies.
  • If you try to enter a term that does not currently exist, you will be given a dialogue box asking if you want to proceed.
    • Strain: existing strains can be viewed in the Strain module.
    • Tissue: a dropdown of tissues currently in the source table aids entry into this field; be sure to tab out to ensure the PWI “remembers” your choice. The entire list of existing tissues can be viewed in the Tissue module.
    • Cell Line: the list of existing cell lines can be found here; not all are used: Existing Cell Lines Report

Related References

There is no specified sort order; they come back in the whatever order the DAO gets them from the database. Thus, the most recently added one will be at the top of the list (under Primary).

Modifying/Updating Antibodies

  • All fields with green labels (antibody attributes) can be updated (e.g. name, genes, references, aliases).

Deleting Antibodies

An antibody that is being used by an assay cannot be deleted. You must delete the assay or modify it so that it uses another antibody first.
