P-WI Antibody Summary Functionality:

  • Displays antibody records associated with any Marker or Reference record.
    • Supported URL for a reference is /pwi/edit/antibodysummary?refs_id=J:(number).
    • Supported URL for a marker is /pwi/edit/antibodysummary?marker_id=MGI:(number)
  • The following fields are displayed verbatim:
    • Antibody MGI ID
    • Antibody Name
    • Antibody Alias(es)
    • Antibody Organism
    • Antibody Type
    • Antibody Class
    • Antibody Notes
    • Antigen Organism
    • Region
    • Notes
    • Markers
    • Reference
  • Sort order is on Marker, Antibody Name, Antibody MGI ID
  • Note that multiple references may be associated with an antibody, but only the Primary reference is displayed on this summary, related references are not shown.
  • The summary returns a maximum of 1,000 rows. If additional rows exist, display them by clicking the "View all..." link above the table, or downloading the tab-delimited file.
