P-WI Assay Detail

  • The Assay Detail page displays data for any assay in the GXD tables - both GXD and Cre assay types.
  • The URL supports /edit/assaydetail/?id=MGI:(number).
  • Thumbnail images, gel assay image labels, and the "Figure Label" column of in situ assays are linked to the appropriate Image Detail page.
  • Conditional genotypes are indicated by text "(conditional)" below the allele pairs for a Specimen (in situs) or a Lane (blots).
  • Age values are displayed as saved in the database. Code does not use logic to translate into abbreviations as on Public.
  • Structure links to EMAPS term and stage in the EMAPA browser
  • Cell Type in results of in situ-type assays links to Cell Type Ontology browser term
  • Each EMAPA/Cell Type combination is one row of the results.
  • Cell Type can be blank.
Notes and other fields are displayed:
  • verbatim
  • superscripted (angle brackets)
  • HTML-enabled
  • notes tag converted

Hover your mouse pointer over the Label to see how the note text is being translated.

Blot Detail Pages

  • Lane information (upper table) - includes label, age, structure, alleles, sex.
  • Lane Notes and Age Notes are displayed at the end of each Lane.
  • Gel Row and Band information (lower table) - includes band size and strength.
  • Row Notes are indicated by an "*" (asterisk) in the Rows and Bands table and written above the table.
  • Band Notes are indicated by a # superscript in the band field. The notes are written below the table. Identical notes are represented by the same number.
  • The band strength 'Not Applicable' is de-emphasized in a gray font.
  • Cell types are not coded in blots
