P-WI (Mapping) Experiment Summary Functionality:

  • Displays mapping experiment records associated with any Marker record.
  • Supported URL for a marker is /pwi/summary/experiment?marker_id=MGI:(number)
  • Only the following Experiment Types are returned:
    • TEXT
    • TEXT-Congenic
    • TEXT-Genetic Cross
    • TEXT-Meta Analysis
    • TEXT-Physical Mapping
    • TEXT-QTL
    • TEXT-QTL-Candidate Genes
  • The summary returns a maximum of 5,000 rows. If additional rows exist, display them by clicking the "View all..." link above the table, or downloading the tab-delimited file.
