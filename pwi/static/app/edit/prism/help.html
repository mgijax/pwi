P-WI PRISM functionality

Basic concepts

  • Image panes belong to a given image, have labels, may or may not have coordinates, and are stored in the database. Panes are listed under their images in the table on the left.
  • Regions (or overlays) are semi-transparent rectangles drawn on top of an actual image, are defined by their coordinates, may be associated with panes, and are ephemeral.
  • PRISM supports curating pane geometry by drawing regions and associating them with panes. PRISM provides several ways to draw panes and several ways to make associations.

Using the tool

  • Load a new J#:
    • Enter the J# (with or without the prefix) in the text field at the top left and hit Return (or tab out of the field). The first image for the J# is opened automatically.
  • See a given figure and its panes:
    • Click on the figure label in the table on the left. The image for the figure is displayed. If pane coordinates have already been defined, the corresponding overlays are also displayed.
  • Draw a region:
    • Drag the mouse over the desired section of the image.
    • Use the Cover feature.
    • Use the OnePane feature. Coordinates are automatically assigned with this feature.
  • Selecting panes and region(s):
    • Both regions and panes can be selected by clicking on them.
      • Clicking a region selects that region and any associated panes.
      • Clicking a pane selects that pane and its associated region, if any. (Other panes associated with the same region, are not selected.)
    • If an image pane has coordinates, then clicking on the pane's row in the table will select the region in the image.
    • To select multiple regions, hold the shift key down while clicking.
    • Clicking the button "SelectAll" will select all regions.
  • Split regions:
    • Select the region(s) to split.
    • To split along a horizontal, Option-click (hold down the Option (or Alt) key while clicking). All selected regions will be split along a horizontal line though the clicked location. (Regions not intersected by the horizontal are unaffected.)
    • To split along the vertical, also hold down the Shift key (Option-Shift-click).
  • Delete region(s) by selecting the region(s) to delete and click the "Delete" button (or type the corresp. keyboard shortcut). Panes that were currently associated with the regions will be dis-associated (and display a "?" for coordinates).
  • Associate a pane with a region:
    • Option 1: Drag the pane label from the table and drop onto the desired region.
    • Option 2: click "OnePane" to associate a single pane with the entire image area. Automatically assigns the coordinates. Use this when the entire image area is the pane.
    • Option 3: Click "ClickAssign" (note: the button stays depressed), and then click the regions in order. Panes will be assigned in order. You may click on a different panelabel (in the table) to start/continue from a different point. When you are done, click "ClickAssign" again, to reset the button.
    • Option 4: Click "GridAssign". Autoassigns all panes in order to defined regions in a left-to-right, top-to-bottom order. Option 4 works better when you have defined regions by splitting rather than by drawing freehand.
  • Associate multiple panes with a region:
    • Select multiple panes in the left column.
    • Drag and drop the selection onto the desired image region.

Error Messages

  • If the J# has no figures in the database (or cannot be found), an error message appears.
  • If a figure record exists, but has no associated image (JPEG file), a "No image available" message appears.
  • If there is no corresponding image, "No image available" is displayed.
  • If you try to leave without saving "There are unsaved changes. Would you like to save your changes?" is displayed.

Mouse actions:

  • click Clicking on a region selects it.
  • shift-click Selects multiple regions.
  • drag Draws a new region.
  • option-click Splits all selected regions along a horizontal line through the click point.
  • option-shift-click Splits all selected regions along a vertical line through the click point.
  • Tip: on a fresh image (no regions yet) option-click or option-shift-click on the image will automatically create a covering region and then do the split.

Tool Bar Buttons:

    Hovering over tool bar buttons shows a description of the tool.
  • Reload
  • PDF
  • Undo
  • Redo
  • -1 0 +1 - zoom in or out
  • Hide - toggles showing the selections
  • Clear - clears all selections
  • Save - saves all figures (not restricted to only the current figure)
  • Cover - covers whole image space
  • Delete - deletes currently selected region.
  • SelectAll
  • ClickAssign - description above
  • GridAssign - description above
  • OnePane - description above

Keyboard Short-cuts

    Note: "Option" may be "Alt" on your keyboard.
  • option-s Save
  • option-z Undo
  • option-r Redo
  • DEL or BACKSPACE Delete selected
  • option-x Delete selected
  • option-shift-x Clear (delete all)
  • option-g GridAssign
  • option-o Onepane
  • option-c Cover
  • - (Just "-", no option key.) Zoom-out
  • + (Just "+", no option key.) Zoom-in
  • 0 (Just "0", no option key.) Original size
