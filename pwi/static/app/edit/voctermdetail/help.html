Vocabulary Term Detail Functionality:

URLs supported include:
  • /pwi/edit/voctermdetail?key=(number)
  • /pwi/edit/voctermdetail?id=MP:(number)
  • /pwi/edit/voctermdetail?id=GO:(number)
  • /pwi/edit/voctermdetail?id=EMAPA:(number)
  • /pwi/edit/voctermdetail?id=EMAPS:(number)
  • /pwi/edit/voctermdetail?id=OMIM:(number)
  • /pwi/edit/voctermdetail?id=DO:(number)
Displays the following data for any term.
  • Term
  • Vocabulary
  • Synonyms
  • Primary ID
  • Secondary (alternate) IDs, if any
  • Definition
  • Comment (public)
  • Number of paths to term
  • Term in DAG context with:
    • relationship type (part-of, is-a, negatively_regulates, etc.)
    • ancestors leading up to root term for the vocabulary
    • siblings
    • immediate child terms
