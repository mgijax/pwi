P-WI Accession Query and Summary functionality:

When a (comma-delimited) list of supported accession IDs is entered, a summary of matching results is returned. Each object is linked to its Detail Page. Because of the variety of object types which may be returned, only the ID and the MGI Type are listed in the summary.

Supported objects and their accession ID types include:
  • Reference - J: number
  • Marker - MGI ID
  • Probe - MGI ID
  • Mapping experiment (only select experiment types) - MGI ID
  • Antibody - MGI ID
  • Assay (GXD or Cre) - MGI ID
  • Image (GXD or Cre) - MGI ID
  • Allele - MGI ID
  • GO term - GO ID
  • MP term - MP ID
  • OMIM term - OMIM ID (numeric value)
  • EMAPA term - EMAPA ID
  • EMAPS term - EMAPS ID
  • AD term - MGI ID
