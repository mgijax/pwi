AD Structure Detail (Vocabulary Term) Detail Functionality:

  • URLs supported include /pwi/detail/adstructure/key/(number) and /pwi/detail/adstructure/MGI:(number).
Displays the following data for any term.
  • Term
  • Synonym(s) - if any
  • MGI ID
  • EMAPS ID
  • EMAPS Term
  • Theiler Stage
  • Notes, if any - these are private and do not display on the Public WI
  • Count of expression results annotated to the term (excludes annotations to child terms) - linked to expression results summary page
  • Term in tree structure with:
    • ancestors leading up to root term
    • siblings
    • immediate child terms
