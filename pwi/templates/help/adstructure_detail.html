AD Structure Detail (Vocabulary Term) Detail Functionality:

  • URLs supported include /pwi/detail/adstructure/MGI:(number).
Displays the following data for any term.
  • Term
  • Synonym(s) - if any
  • MGI ID
  • EMAPS ID
  • EMAPS Term
  • Theiler Stage
  • Notes (private), if any (need to double-check on this. any examples?)
  • Count of expression results annotated to the term (excludes child terms) - linked to expression results summary page
  • Term in tree structure with:
    • ancestors leading up to root term
    • siblings
    • immediate child terms
