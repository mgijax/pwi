AD Structure Query and Summary Results Functionality:

AD Structure Summary Results URLs supported include:
  • /pwi/summary/adstructure?structure_text=(inputString)&theiler_stages=(value)&theiler_stages=(value)...
  • /pwi/summary/adstructure?structure_text=(inputString) - if only text for searching in Structures is submitted.
  • /pwi/summary/adstructure?structure_text=&theiler_stages=(value) - if a single Theiler Stage is submitted.
Functionality:
  • The "You Searched For" section echoes any Structures input text string and Theiler Stage(s) from the user's query.
  • Text searching is "contains". Searching for "ear" will also return "heart".
  • The input string is searched in Structure terms and Synonyms. Matching text is displayed in boldface wherever it occurs (Structure or Synonym).
  • Synonyms are displayed for all Structures returned, even if the Synonym does not contain text matching the search string.
  • Results are returned ordered by (1) Theiler Stage, (2) Structure (alphabetical order).
  • The MGI ID of the Structure is linked to an AD Structure (vocabulary term) Detail page.
  • A "Search Again" input form appears on every Summary and Detail page to allow searching for a new text string. Text is searched in all Theiler Stages when using this form.
