P-WI Allele Editor Interface functionality

General Description

The Allele module is a search, display and edit page all in one. A user must be logged in to edit but can search without logging in.

Keyboard Short-cuts

There are reported issues using these short-cuts with Mac/Firefox.
  • Search = Ctrl-Alt-S
  • Clear = Ctrl-Alt-C
  • Modify = Ctrl-Alt-M
  • Previous = Ctrl-Alt-P
  • Next = Ctrl-Alt-N

Color-coded Labels

  • Thistle = search only
  • Green = search & edit
  • Blue = read-only
  • Yellow = current row

Supported Browsers

  • Chrome and FireFox are supported on both PC and Mac computers.
  • Safari is NOT supported for the PWI. There are features that do not work.

Searching

All fields that have labels highlighted in blue/green may be used for searching. They may be searched individually or in combination. To include data in a tab in a search: click on the tab to expand it and enter text to be searched. When more than one field is used in a search, it is treated as an AND search.

Wildcards

  • Wildcards (%) may be used for most fields.
  • A wildcard may be used at the beginning, middle and/or end of your search text: e.g. %paired%, casein% or AK32%43%.

Search by Dates

  • Valid date formats for searching are:
    • mm/dd/yyyy or mm-dd-yyyy
    • mm/dd/yy or mm-dd-yy
    • yyyy/mm/dd or yyyy-mm-dd
  • Searching for a range of dates:
    • equal to: enter a single date
    • greater than: > mm/dd/yyyy
    • greater than or equal to: >= mm/dd/yyyy
    • less than: < mm/dd/yyyy
    • less than or equal to: <= mm/dd/yyyy
    • between two dates (inclusive): mm/dd/yyyy..mm/dd/yyyy

Search Data

When you click on the Search button:

  • the results are displayed in a table with each row representing a single allele.
  • Results are displayed by status (in progress, deleted, and reserved prior to approved or autoloaded) and in alpha-numeric order.

Add New Data

Fields required to add a new allele or transgene include:

  • symbol
  • allele name
  • marker (required for approved alleles)
  • original reference
  • generation type
  • germ line transmission status

Other fields have defaults such as:

  • status (reserved)
  • inheritance mode (not applicable)
  • collection (not specified)
  • mixed (no)
  • extinct (no)
  • strain of origin (blank for all generation types except gene trapped and targeted that default to not specified).
  • The mutant cell line is blank for all generation types except gene trapped and targeted that default to not specified.

Once required data has been entered, select ‘Add’ in the top right menu.

Modify Existing data

  • Fields with green headers can be modified.
  • Be sure to select ‘Modify’ in the top right menu to save changes.
  • Warning: approved alleles must have approved an approved marker and original reference (not J:23000).
