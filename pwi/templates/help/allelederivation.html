P-WI Allele Derivation Editor Interface functionality

General Description

The Allele Derivation module is a search, display and edit page all in one. A user must be logged in to edit but can search without logging in.

Keyboard Short-cuts

There are reported issues using these short-cuts with Mac/Firefox.
  • Search = Ctrl-Alt-S
  • Clear = Ctrl-Alt-C
  • Modify = Ctrl-Alt-M
  • Previous = Ctrl-Alt-P
  • Next = Ctrl-Alt-N

Color-coded Labels

  • Thistle = search only
  • Green = search & edit
  • Blue = read-only
  • Yellow = current row

Supported Browsers

  • Chrome and FireFox are supported on both PC and Mac computers.
  • Safari is NOT supported for the PWI. There are features that do not work.

Searching

All fields that have labels highlighted in blue/green may be used for searching. They may be searched individually or in combination. To include data in a tab in a search: click on the tab to expand it and enter text to be searched. When more than one field is used in a search, it is treated as an AND search.

Search by Dates

  • Valid date formats for searching are:
    • mm/dd/yyyy or mm-dd-yyyy
    • mm/dd/yy or mm-dd-yy
    • yyyy/mm/dd or yyyy-mm-dd
  • Searching for a range of dates:
    • equal to: enter a single date
    • greater than: > mm/dd/yyyy
    • greater than or equal to: >= mm/dd/yyyy
    • less than: < mm/dd/yyyy
    • less than or equal to: <= mm/dd/yyyy
    • between two dates (inclusive): mm/dd/yyyy..mm/dd/yyyy

Adding a derivation

  • Minimal information required to add a new derivation includes: Derivation Type (select from list), Creator (select from list), Vector name (type in exactly or select from the list on the right below PCL), and Vector Type (select from list). Additionally, PCL can be typed in exactly or selected from the list on the right.
  • Alternative, an existing derivation can be searched for then deselected, modified and added as a new derivation.
  • The J# is not required to add a derivation. The field remains bank if no J# is entered.
  • If the derivation already exists, you will receive a warning to that effect and the duplicate derivation will not be added.

Modifying a derivation

  • Fields that can be modified within the field: J#, Creator, Vector Name (must match one on pick list), Vector Type. Note that PCL, PCL strain and Cell Line Type can only be modified by selecting a different PCL from the menu on the right. Note that you cannot type in and modify thistle colored boxes.

Deleting a derivation

  • Attempting to delete a derivation associated with one or more mutant cell lines is not allowed and will generate a warning. First removed mutant cell lines associated with the derivation in order to be allowed to delete the derivation.
  • For more information, see http://mgiprodwiki.jax.org/mediawiki/index.php/cur:Allele_Derivations
