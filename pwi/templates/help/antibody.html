P-WI Antibody Editor Interface functionality

General Description

The Antibody module is a display and edit page all in one. A search with no parameters returns all antibodies. You may add/edit/delete antibodies and their references, markers and aliases.You may add/update the antibody with an existing antigen, but must go to the antigen module to add new or update existing antigens. A user must be logged in to edit but can search without logging in.

  • Thistle = search only
  • Green = edit
  • Blue = read only
  • Yellow = current row

Supported Browsers

  • Chrome and FireFox are supported on both PC and Mac computers.
  • Safari is NOT supported for the PWI. There are features that do not work.
