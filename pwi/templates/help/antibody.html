P-WI Antibody Editor Interface functionality

General Description

The Antibody module is a display and edit page all in one. A user must be logged in to edit but can search without logging in.

Keyboard Short-cuts

  • Add = Ctrl-Alt-A
  • Delete = Ctrl-Alt-D
  • Search = Ctrl-Alt-S
  • Clear = Ctrl-Alt-C
  • Modify = Ctrl-Alt-M
  • First = Ctrl-Alt-F
  • Previous = Ctrl-Alt-P
  • Next = Ctrl-Alt-N
  • Last = Ctrl-Alt-L

Color-coded Labels

  • Thistle = search only
  • Green = edit
  • Blue = read only
  • Yellow = current row

Supported Browsers

  • Chrome and FireFox are supported on both PC and Mac computers.
  • Safari is NOT supported for the PWI. There are features that do not work.

Search by Dates

  • Valid date formats for searching are:
    • mm/dd/yyyy or mm-dd-yyyy
    • mm/dd/yy 0r mm-dd-yy
    • yyyy/mm/dd or yyyy-mm-dd
  • Searching for a range of dates:
    • equal to: enter a single date
    • greater than: > mm/dd/yyyy
    • greater than or equal to: >= mm/dd/yyyy
    • less than: < mm/dd/yyyy
    • less than or equal to: <= mm/dd/yyyy
    • between two dates (inclusive): mm/dd/yyyy..mm/dd/yyyy

Searching

  • All fields may be used for searching.
  • They may be searched individually or in combination. When more than one filed is used in a search, it is treated as an AND search.
  • Antigen fields can only be used for searching, creating/modifying an antigen must be done in the antigen module.
  • A search of the Name field will simultaneously search Names and Aliases. A search of the Alias field will only search Aliases.
  • A search with no parameters returns all antibodies.

Wildcards

  • Wildcards (%) may be used for most fields.
  • A wildcard may be used at the beginning, middle and/or end of your search test: e.g. %paired%, casein% or AK32%43%.

Creating New Antibodies

  • The fields required to create an antibody are: Name, Antigen ID, and Primary Reference. All other fields are optional.
  • One antigen can be associated with >1 antibody.
  • Each antibody must have one primary reference; all other references should be type = related.
  • Aliases are not required to have a J#; they may be entered with out.
  • There are antibodies used in assays that do not have associated genes. These antibodies recognize non-mouse genes, e.g. B-galactosidase and GFP, and are used in knock-in and transgenic in situ reporter assays, as well as in recombinase reporter assays.

Modifying/Updating Antibodies

  • All fields with green labels (antibody attributes) can be updated (e.g. name, antigen id, genes, references, aliases).
  • Antigen attributes (purple labels) can only be updated in the antigen module.

Deleting Antibodies

An antobody that is being used by an assay cannot be deleted. You must delete the assay or modify it so that it uses another antibody first.
