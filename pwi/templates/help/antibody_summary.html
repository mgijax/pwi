P-WI Antibody Summary Functionality:

  • Displays antibody records associated with any Marker or Reference record.
    • Supported URL for a reference is /pwi/summary/antibody?refs_id=J:(number).
    • Supported URL for a marker is /pwi/summary/antibody?marker_id=MGI:(number)
  • The following fields are displayed verbatim:
    • Antibody Name
    • Antibody Alias(es)
    • Antibody Notes
    • Antibody Name
    • Antigen Name
  • Note that multiple references may be associated with an antibody, but only 1 is displayed on this summary.
  • The summary returns a maximum of 5,000 rows. If additional rows exist, display them by clicking the "View all..." link above the table, or downloading the tab-delimited file.
