P-WI Assay Detail functionality:

  • The Assay Detail page displays data for any assay in the GXD tables - both GXD and Cre assay types.
  • The URL supports /detail/assay/MGI:(number) or /detail/assay/key/(number).
  • Thumbnail images, gel assay image labels, and the "Figure Label" column of in situ assays are linked to the appropriate Image Detail page.
  • Conditional gentoypes are indicated by text "(conditional)" below the allele pairs for a Specimen (in situs) or a Lane (blots).
  • Age values are displayed as saved in the database. Code does not use logic to translate into abbreviations as on Public.
Notes fields:
  • Three notes fields are displayed "translated" as they would be on public for GXD pages and for Cre pages. Hover your mouse pointer over the Label to see how the note text is being translated (verbatim, "HTML-enabled" (interprets angle brackets as HTML tags), MGI-markup (interprets angle brackets as indicating superscript) and/or using Notes Tag converter).
    • Assay Notes
    • Specimen Notes
    • Result Notes
  • Notes translated in the format of Cre Public WI pages are displayed in gray font color.
  • Blot data are displayed as simply as possible. Notes data are not associated with a "footnote" but are displayed in separate tables below the results table. These include:
    • Age Notes
    • Row Notes
    • Band Notes
  • Blot lane labels are translated from MGI markup (angle brackets for superscript). Code uses logic to supply lane numbers when no text label is found. Numbers correspond to "Lane" numbers in Notes below the results table.
  • When 2 or more rows are present, blot rows are numbered in results table column headers. Numbers correspond to "Gelrow" numbers in Notes below the results table.
  • A lane having one or more Band Notes is indicated by an asterisk in a column between the Lane Note and Age Note columns. Band Notes are displayed in a table below the results table.
