P-WI Assay Detail functionality:

  • The Assay Detail page displays data for any assay in the GXD tables - both GXD and Cre assay types.
  • The URL supports /detail/assay/MGI:(number) or /detail/assay/key/(number).
  • Three notes fields are displayed "translated" as they would be on public for GXD pages and for Cre pages. Hover your mouse pointer over the Label to see how the note text is being translated (verbatim, "HTML-enabled" (interprets angle brackets as HTML tags), MGI-markup (interprets angle brackets as indicating superscript) and/or using Notes Tag converter).
    • Assay Notes
    • Specimen Notes
    • Result Notes
  • Notes translated in the format of Cre Public WI pages are displayed in gray font color.
  • Blot data are displayed as simply as possible. Most notes data are not associated with a "footnote" but are displayed (for now) in a separate table. These include:
    • Age Notes
    • Row Notes
    • Band Notes
  • Blot lane labels are translated from MGI markup (angle brackets for superscript). Code uses logic to supply lane numbers when no label is found.
