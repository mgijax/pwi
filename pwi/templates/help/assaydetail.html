P-WI Assay Detail

  • The Assay Detail page displays data for any assay in the GXD tables - both GXD and Cre assay types.
  • The URL supports /assaydetail/MGI:(number) or /assaydetail/key/(number).
  • Thumbnail images, gel assay image labels, and the "Figure Label" column of in situ assays are linked to the appropriate Image Detail page.
  • Conditional gentoypes are indicated by text "(conditional)" below the allele pairs for a Specimen (in situs) or a Lane (blots).
  • Age values are displayed as saved in the database. Code does not use logic to translate into abbreviations as on Public.
Notes fields:
  • Three notes fields are displayed "translated" as they would be on public for GXD pages and for Cre pages. Hover your mouse pointer over the Label to see how the note text is being translated (verbatim, "HTML-enabled" (interprets angle brackets as HTML tags), MGI-markup (interprets angle brackets as indicating superscript) and/or using Notes Tag converter).
    • Assay Notes
    • Specimen Notes
    • Result Notes
  • Notes translated in the format of Cre Public WI pages are displayed only for Cre assay types.
  • Blot data and Notes are displayed as simply as possible:
    • Lane labels, if any, are translated from MGI markup (angle brackets for superscript).
    • Lanes are numbered in the results table. Rows are numbered when 2 or more exist.
    • Lane Notes and Age Notes are displayed at the end of each Lane in the results table.
    • Row Notes and Band Notes are displayed in separate tables below the results table.
    • The presence of one or more Band Notes for a Lane is indicated by an "*" (asterisk) in the appropriate Lane in the results table.
    • Lane and Row numbers in the results table are displayed as Lane # and Gelrow # with any Notes. Lane and Row label text, if any, is shown in boldface.
