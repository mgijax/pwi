GXD HT Experiment Load
  • This is an incremental load of high-throughput experiments from ArrayExpress. These are being loaded for the purpose of indexing some of them using standard vocabularies for the meta data of the experiment. It is expected that users will use GXD to find interesting experiments, but will go to the source for the details of the experiment itself.
    • Only experiments for mouse are loaded.
    • The load runs on a regular schedule (TBD - monthly/weekly?).
    • Newly loaded experiments will be set to "Not Evaluated".
    • At this point there are no plans to delete any of the loaded experiments even if curators flag them with a "No" (i.e. don't index this one).
