GXD Index Summary functionality:

  • Uses the Assay Type abbreviations found in the database (and the EI).
  • "Counts of Indexed Genes By Age and Assay" table and "Matching Index Records" tables display only rows and columns which have data for the reference and gene.
  • "Matching Index Records" tables include Notes text, if any.
for a Reference
  • Supported URL format is /pwi/summary/gxdindex?refs_id=J:(number)
  • "Matching Index Records" table includes text "(Fully Coded)" following the gene name if the reference + marker is fully coded.
  • Includes the Priority score and Conditional menu choice.
for a Marker (Gene)
  • Supported URL format is /pwi/summary/gxdindex?marker_id=MGI:(number)
  • References table lists all references indexed to the gene, the Priority score and the Conditional menu choice. Fully Coded references are indicated by an asterisk (*).
