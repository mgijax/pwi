GXD Index Summary for a Reference functionality:

  • Supported URL format is /pwi/summary/gxdindex?refs_id=J:(number)
  • Index Records tables use the Assay Type abbreviations found in the database (and the EI).
  • Each gene's Index Records table displays only rows and columns which have data.
  • Includes text "(Fully Coded)" following the gene name if the reference + marker is fully coded.
  • Includes Notes text, if any.
  • "Counts of Indexed Genes By Age and Assay" counts are not yet linked to an Index Summary for Reference+Assay+Age page.
