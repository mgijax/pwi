P-WI Image Detail Functionality:

  • Displays images associated with any Expression assay: GXD or Cre.
  • Supported URL format is /pwi/detail/image/MGI:(number) using the MGI ID of the image.
  • Images are displayed at the full size uploaded.
  • Links in the "Copyright" data are displayed with P-WI external link formatting.
  • When present, links to GenePaint or GUDMAP are displayed under "Other Database Links".
  • The "Elsevier" tag, "DXDOI" tag, and other tags in the "Copyright" data are translated and displayed as on Public.
  • Although Molecular and Phenotype image IDs will allow the page to render, not all associated data will be present.
