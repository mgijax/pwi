P-WI Image Detail Functionality:

  • Displays images associated with any Expression assay: GXD or Cre.
  • Supported URL formats are /pwi/detail/image/MGI:(number) using the MGI ID of the image and /pwi/detail/image/key/(number).
  • Images are displayed at the full size uploaded.
  • When present, links to GenePaint or GUDMAP are displayed under "Other Database Links".
  • Assay Data table order:
    • Rows are ordered alphanumerically by pane label.
    • Assay & Gene data for a pane are ordered by Assay ID, ascending (roughly, from first entered to last entered).
  • Links in the "Copyright" data are displayed with P-WI external link formatting.
  • The "Elsevier" tag, "DXDOI" tag, and other tags in the "Copyright" data are translated and displayed as on Public.
  • NOTE: Although Molecular and Phenotype image IDs will allow the page to render, not all associated data will be present.
