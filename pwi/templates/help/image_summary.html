P-WI Image Summary functionality:

  • The page includes all images associated with an allele, whether directly associated or associated through a genotype in which the allele participates.
  • For a given allele, the following image types are displayed:
    • class = molecular
    • class = phenotypes
  • Images are sorted by class (molecular before phenotypes), then by primary ("yes" before "no"), then by database _image_key descending (to put newest images before older ones).
  • For each image set, the thumbnail image is shown, followed by the fullsize image.
  • Thumbnail images are shown at actual size. Fullsize images are scaled down so that the width never exceeds 400px. Actual dimensions are stated.
  • This page is accessed only via the link on an Allele Detail Page. The URL supports /pwi/summary/image?allele_id=MGI:(number). An Image Summary for an allele is the only Image Summary type supported; no other Image Summary type pages exist.
  • The Caption field does the following:
    • Use Notes Tag Converter for \AlleleSymbol tags.
    • Interpret HTML for <sup> tags and others.
    • B2B alleles include HTML with javascript to play movie from YouTube in the page. These are handled as they are on the Public Image Detail page: clicking the "Play Video" link fires the javascript code to replace the image file with the YouTube player and plays the designated video within the page at the designated size.
  • The Copyright field does the following:
    • Interpret HTML for <a> tags (and others?).
    • Translate any Notes Tags (e.g., \JBiolChem(8663429|JBC|), \JLipidRes(10884287|JLR|), etc.)
Associated Data:
  • All alleles associated with each image.
