P-WI Imagepane Summary functionality:

  • Includes all image panes for GXD and Cre (not Pheno) for a given reference. Includes both in situ and blot panes. There are no special Cre sorts.
  • There is no row limit.
  • Table Data
    • Column 1: MGI id of image sub. Image pane displayed based on coordinates. Display size is based on a percentage of the pane size. Linked to the image detail page. If there is no image, only the ID of the image stub shows.
    • Column 2: Figure label (primary sort; in numerical order).
    • Column 3: Pane (secondary sort; in apphabetical order; blank if no pane label)
    • Column 4: Specimen label; multiples separated by line breaks; blank if annotated to blot.
    • Column 5: Assay Type; multiples separated by line break.
    • Column 6: MGI id of annotated assay(s); gene of assay in parens after ID; multiples separated by a link break; blank if not annotated to; (tertiary sort; in alphabetical order by gene).
