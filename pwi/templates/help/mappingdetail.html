P-WI (Mapping) Experiment Detail Functionality:

  • Supported URL is /pwi/edit/mappingdetail?id=MGI:(number).
  • Only the following fields are displayed:
    • MGI ID
    • Reference
    • Reference Note - displayed verbatim as on Public WI
    • Experiment Type - see Experiment Summary page functionality description for supported Experiment Types
    • Chromosome
    • Notes - displayed verbatim as on Public WI
    • Data grid from the top of the EI Mapping Module, containing the following fields:
      • Marker
      • Marker ID
      • Allele - with angle brackets converted to superscript as on Public WI
      • Assay Type
      • Description
      • Matrix Data
    • Additional data fields which may exist for other experiment types are not returned.
