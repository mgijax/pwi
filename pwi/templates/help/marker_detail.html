P-WI Marker Detail functionality:

  • The Marker Detail page displays data for any marker in the Marker tables.
  • The URL supports /detail/marker/MGI:(number) or /detail/marker/key/(number).
  • Mouse markers have MGI IDs and database key values.
  • Markers for other organisms have database key values but not MGI IDs.
Summary Links:
  • Alleles - includes all private and public alleles associated directly with the marker. Does not include "expresses" or other relationships.
  • References - includes references to which the marker is directly associated with the value "General" or "Strain Specific" as well as references associated with the marker via alleles, GO annotations, GXD Lit Index, probes, etc.
  • GXD/Cre Assays - includes all GXD and Cre assays where the marker is the Assayed Gene.
  • GXD/Cre Results - includes all GXD and Cre assay results where the marker is the Assayed Gene.
  • Probes
  • Antibodies
  • Mapping Experiments - includes only specific experiment types.
