P-WI Marker Query and Summary functionality:

The Marker Query and Summary page returns markers from the Marker tables matching your text in the Name, Symbol, or Synonyms fields for Mouse records. It does not search other organisms (e.g., no Human nomenclature). It does not search Allele nomenclature.

  • official
  • reserved

Sort order: (1)Status, (2)Symbol in ASCII order.

Only the first 5,000 matching records are displayed. To get all results if more than 5,000 records match, click the "View All Matches" link at the top of each table, or download the tab-delimited file.

When querying by Nomenclature, each item that includes text matching the query is highlighted in boldface.

When querying using Feature Type:

  • Query results include all subterms of the checked term(s) - e.g., checking "gene" also includes its 17 subterms.
Marker Summary for a Reference functionality:
  • This summary includes Markers where Reference association type is "General" or "Strain Specific". It also includes Markers indirectly associated to a Reference through assays, GO annotations, alleles, mapping experiments, or other objects.
  • The URL is /pwi/summary/marker?refs_id=J:(number)
