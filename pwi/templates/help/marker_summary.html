P-WI Marker Query and Summary functionality:

The Marker Query and Summary page returns Mouse markers from the Marker tables matching your text in the Name, Symbol, or Synonyms fields.

It also returns markers from the Nomen tables that match your text in the Name, Symbol, or Synonyms fields and have one of the following status values:

  • in progress
  • deleted
  • reserved
  • approved

Only the first 5,000 matching records are displayed. To get all results if more than 5,000 records match, click the "View All Matches" link at the top of each table.

When querying by Nomenclature, each item that includes text matching the query is highlighted in boldface.

Marker Summary for a Reference functionality:

This summary includes only Markers where Reference was "Used" for direct annotation.
