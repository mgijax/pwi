P-WI MP Annotations Editor Interface functionality

General Description

The MP Annotations module is a search, display and edit page all in one. A user must be logged in to edit but can search without logging in.

Keyboard Short-cuts

There are reported issues using these short-cuts with Mac/Firefox.
  • Search = Ctrl-Alt-S
  • Clear = Ctrl-Alt-C
  • Modify = Ctrl-Alt-M
  • Previous = Ctrl-Alt-P
  • Next = Ctrl-Alt-N

Color-coded Labels

  • Thistle = search only
  • Green = search & edit
  • Blue = read only
  • Yellow = current row

Supported Browsers

  • Chrome and FireFox are supported on both PC and Mac computers.
  • Safari is NOT supported for the PWI. There are features that do not work.

Searching

All fields that have labels highlighted in blue/thistle may be used for searching. They may be searched individually or in combination. To include data in a tab in a search: click on the tab to expand it and enter text to be searched. When more than one field is used in a search, it is treated as an AND search.

Wildcards

  • Wildcards (%) may be used for most fields.
  • A wildcard may be used at the beginning, middle and/or end of your search text: e.g. %paired%, casein% or AK32%43%.

Search by Dates

  • Valid date formats for searching are:
    • mm/dd/yyyy or mm-dd-yyyy
    • mm/dd/yy or mm-dd-yy
    • yyyy/mm/dd or yyyy-mm-dd
  • Searching for a range of dates:
    • equal to: enter a single date
    • greater than: > mm/dd/yyyy
    • greater than or equal to: >= mm/dd/yyyy
    • less than: < mm/dd/yyyy
    • less than or equal to: <= mm/dd/yyyy
    • between two dates (inclusive): mm/dd/yyyy..mm/dd/yyyy

Search by Genotype

The Genotype field includes the strain background followed by the allele pairs(e.g. 129S/SvEv(FVB)-Thbd<tm1.1(THBD)Sltz>TThbd<tm1.1(THBD)Sltz>, Thbd<tm1.1(THBD)Sltz>). Be sure to include wildcards when not entering a full genotype statement. The search results will present the genotypes specified by the search criteria. The main portion of the page displays the MP annotations associated with the selected genotype (denoted by a yellow highlight). Use the mouse or First/Next/Previous/Last buttons to change the selected genotype and MP annotation display.)

Qualifier

The 'Norm' qualifier should only be associated with MP header term. All other MP terms default to (none).

Evidence Code

The default evidence code is 'EXP'. 'TAS' should only be used when the information associated with the MP term is found within the reference but attributable to another source that cannot be obtained.

Sex

'NA' is the default sex. Switch to F or M as necessary.

Dialogue warning boxes (popups) behavior

  • 'Cannot save this Annotation if a record is not selected.' Click 'OK' and select a genotype record.
  • 'Invalid MP Acc ID: MP:MP:0002169' Click 'OK' and correct invalid MP terms.
  • 'Invalid Reference: J:87272f' Click 'OK' and correct invalid J numbers.
  • 'This reference is not associated to all Alleles of this Genotype. Do you want the system to add a 'Used-FC' reference association for these Alleles?' To add the J number to the alleles associated with the selected genotype, click 'OK'. To prevent the addition, click 'cancel' and edit J number.
  • 'Duplicate Order Detected in Table. Cannot Modify.' Click 'OK' and correct duplication of header order numbers.

Entering MP annotations

Search for relevant genotype(s) by MGI ID or genotype (remember to use wildcards when incomplete information is entered). Paste MP term number (e.g. MP:0005375). Qualifier defaults to 'none' for all terms except header system terms. Enter J number (J:123456). Tabing through the field or selecting the next MP annotation row will autofill the J number field with the J# of the previous MP annotation row. The evidence code defaults to EXP when annotation is saved. Sex defaults to NA when annotation is saved. Save using the "Save annotations, Notes, Header Order" button under the "Search" button.

Entering Notes

Select an MP annotation row and add a row in the Notes section below the MP annotation table. Note type defaults to General. Select 'normal' or 'background sensitivity' as needed. Save using the "Save annotations, Notes, Header Order" button under the "Search" button.

Editing Header

The table will display the headers for the MP terms annotated. Use the up and down toggles next to the order number to alter the order of header terms. A warning will be triggered if order numbers are duplicated. Be sure not to skip a number as that will not be detected. Save using the "Save annotations, Notes, Header Order" button under the "Search" button.

Clipboard

The clip board is used to copy one or more MP annotations (MP term and Sex) from one genotype to another. To copy one or more MP annotations individually, selected the MP row(s) and clicking 'copy selected annotations'. The MP term number, Sex, and MP term name will appear on the clipboard. Repeat for all MP terms needed. To select all MP annotations associated with a genotype, click 'Copy all annotations to clipboard'. Once desired terms are copied, select the desired genotype from the results selection and click 'Paste clipboard annotations to table' to add the MP term and sex to the desired genotype. J number will need to be added prior to saving the annotation. Clipboard will not reset when searching. Delete removes selected clipboard annotation. Clear removes all annotations on the clipboard.
