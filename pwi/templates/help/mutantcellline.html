P-WI Mutant Cell Line Editor Interface functionality

General Description

The Mutant Cell Line module is a search, display and edit page all in one. A user must be logged in to edit but can search without logging in.

Keyboard Short-cuts

There are reported issues using these short-cuts with Mac/Firefox.
  • Search = Ctrl-Alt-S
  • Clear = Ctrl-Alt-C
  • Modify = Ctrl-Alt-M
  • Previous = Ctrl-Alt-P
  • Next = Ctrl-Alt-N

Color-coded Labels

  • Thistle = search only
  • Green = search & edit
  • Blue = read-only
  • Yellow = current row

Supported Browsers

  • Chrome and FireFox are supported on both PC and Mac computers.
  • Safari is NOT supported for the PWI. There are features that do not work.

Searching

All fields that have labels highlighted in blue/green may be used for searching. They may be searched individually or in combination. To include data in a tab in a search: click on the tab to expand it and enter text to be searched. When more than one field is used in a search, it is treated as an AND search.

Search by Dates

  • Valid date formats for searching are:
    • mm/dd/yyyy or mm-dd-yyyy
    • mm/dd/yy or mm-dd-yy
    • yyyy/mm/dd or yyyy-mm-dd
  • Searching for a range of dates:
    • equal to: enter a single date
    • greater than: > mm/dd/yyyy
    • greater than or equal to: >= mm/dd/yyyy
    • less than: < mm/dd/yyyy
    • less than or equal to: <= mm/dd/yyyy
    • between two dates (inclusive): mm/dd/yyyy..mm/dd/yyyy

Search by Mutant Cell Line

  • Type mutant cell line using wild card character (%) if needed.

Search by Derivation or Derivation component

  • All mutant cell lines associated with an individual derivation can be searched by selecting an existing derivation from the right hand side menu.
  • All mutant cell lines associated with one or more derivation Creator, Cell Line Type, Derivation type, or Vector Name.

Search by Allele Symbol(s)

  • Enter an existing allele symbol. Multiple symbols should be separated by a comma and space.

Search by Stem Cell Line

  • Select a Stem Cell Line from the right hand menu.

Search by Vector Name

  • Select Vector Name form the right hand menu.

Adding a new Mutant Cell Line

  • Enter Mutant Cell Line and select a Derivation from the right hand menu.

Modifying an existing Mutant Cell Line

  • Search for the Mutant Cell Line to modify. The only modifications possible on this page are changing the Mutant Cell Line name or the derivation associated with Mutant Cell Line. To change aspects of the derivation (Creator, Parent Cell Line, Cell Line Type, Derivation Type, Vector Name or Vector Type). Allele associations with a Mutant Cell Line can only be changed in the Allele module.

Deleting an existing Mutant Cell Line

  • Mutant Cell Lines can be deleted if no allele association exists. Remove the Mutant Cell Line in the allele module, then proceed with deleting the Mutant Cell Line.
