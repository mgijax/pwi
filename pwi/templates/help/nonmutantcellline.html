P-WI Non-Mutant Cell Line Editor Interface functionality

General Description

The Non-Mutant Cell Line module is a search, display and edit page all in one. A user must be logged in to edit but can search without logging in. Modifying/Adding lines through this module will modify and add to the Parent Cell Line list used by various other pages. See the worksheet of standard and non-standard PCLs.

Keyboard Short-cuts

There are reported issues using these short-cuts with Mac/Firefox.
  • Search = Ctrl-Alt-S
  • Clear = Ctrl-Alt-C
  • Modify = Ctrl-Alt-M
  • Previous = Ctrl-Alt-P
  • Next = Ctrl-Alt-N

Color-coded Labels

  • Thistle = search only
  • Green = search & edit
  • Blue = read-only
  • Yellow = current row

Supported Browsers

  • Chrome and FireFox are supported on both PC and Mac computers.
  • Safari is NOT supported for the PWI. There are features that do not work.

Searching

All fields that have labels highlighted in blue/green may be used for searching. They may be searched individually or in combination. To include data in a tab in a search: click on the tab to expand it and enter text to be searched. When more than one field is used in a search, it is treated as an AND search.

Search by Dates

  • Valid date formats for searching are:
    • mm/dd/yyyy or mm-dd-yyyy
    • mm/dd/yy or mm-dd-yy
    • yyyy/mm/dd or yyyy-mm-dd
  • Searching for a range of dates:
    • equal to: enter a single date
    • greater than: > mm/dd/yyyy
    • greater than or equal to: >= mm/dd/yyyy
    • less than: < mm/dd/yyyy
    • less than or equal to: <= mm/dd/yyyy
    • between two dates (inclusive): mm/dd/yyyy..mm/dd/yyyy

Searching

  • Search by Mutant Cell Line or Cell Line Strain
  • Enter Non-Mutant Cell Line name or strain using wild-card (%) as needed.

Modifying

  • Modify Mutant Cell Line
  • Modify Non-Mutant Cell Line, Cell Line Strain and Cell Line Type and select modify.

Adding Data

  • Add Mutant Cell Line
  • Enter Non-Mutant Cell Line, Cell Line Strain and Cell Line Type and select add.
