P-WI Probe Detail Functionality:

  • Displays molecular probe/clone data.
  • Includes all fields from the EI unless stated below.
  • Names are displayed verbatim.
  • Primers include fewer fields than cDNA, genomic, and other probe types.
  • Raw Sequence Notes are not displayed.
Associated Data
  • Other Accession IDs - from other providers
  • Markers - associated with the probe record
  • References
    • Data associated with a probe reference will be added later (aliases, DNA sequence IDs, Endonucleases (RFLP data)).
    • Notes fields associated with a probe reference are not displayed.
  • Other Summary Links
    • Expression Assays - assay records with which the probe is associated
