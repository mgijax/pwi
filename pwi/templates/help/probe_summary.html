P-WI Probe Summary Functionality:

  • Displays probe records associated with any Marker or Reference record.
  • Sort order: (1) Probe name.
  • Multiple markers may be associated with any probe. All are displayed. Markers with a "P" relationship display "PUTATIVE".
  • Supported URL for a reference is /pwi/summary/probe?refs_id=J:(number).
  • Supported URL for a marker is /pwi/summary/probe?marker_id=MGI:(number)
  • Only 2,500 rows are returned for a summary. If additional rows exist, click the "View All Probes" link at the top of the table, or download the tab-delimited file.
