P-WI Reference Query and Summary functionality:

The Reference Summary does not include special data fields associated with BOOKS.

Sort order: database key, descending (reverse order of being added to the database).

Only the first 250 matching records are displayed. If additional rows exist, click the "View All Reference Matches" link at the top of the table, or download the tab-delimited file.

Associated Data:
  • Alleles: includes all Alleles.
  • Markers: includes all Markers:
    • where Reference relationship type is "General" or "Strain Specific".
    • indirectly associated to the Reference through other objects including assays, alleles, GXD lit index, GO annotations, probes, nomen, and mapping data.
    • Does not include associations to GO annotated terms themselves.
  • GXD Lit Index Records (summary of all indexed markers)
  • GXD/Cre Assays: includes all GXD expression assays and Cre activity assays associated with the reference.
  • GXD/Cre Results: includes all GXD expression assay results and Cre activity assay results associated with the reference.
  • Antibodies
  • Probes (molecular segments)
  • Specimens: includes all GXD specimen information associated with in situ type assays in the reference.
  • Exp Images: includes all GXD images/panes associated with the reference.
  • Mapping experiments - mapping data summaries exist only for a marker
Reference Summary for MGI Object
  • For a marker, the supported URL is /pwi/summary/reference?marker_id=MGI:(number)
  • For an allele, the supported URL is /pwi/summary/reference?allele_id=MGI:(number)
