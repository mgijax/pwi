P-WI Reference Query and Summary functionality:

The Reference Query and Summary does not include special data fields associated with BOOKS.

References are sorted by database key, descending (in lieu of publication date).

Only the first 100 matching records are displayed. To get all results if more than 100 records match, click the "View All Reference Matches" link at the top of the table.

Associated Data:
  • Markers: includes only Markers where Reference was "Used" for direct annotation.
  • Expression Assays: includes all GXD expression assays and Cre activity assays associated with the reference.
