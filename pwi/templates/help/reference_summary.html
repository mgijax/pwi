P-WI Reference Query and Summary functionality:

The Reference Summary does not include special data fields associated with BOOKS.

Sort order: database key, descending (reverse order of being added to the database).

Only the first 250 matching records are displayed. To get all results if more than 250 records match, click the "View All Reference Matches" link at the top of the table.

Associated Data:
  • Markers: includes only Markers where Reference type is "General" (e.g. J:203421 has 4 markers in the P-WI but 21 on Public due to GXD Lit Index and other records).
  • Alleles: includes only Alleles where Reference type is "Indexed", "Original", "Molecular", "Expression only", "Transmission", "Used-FC" (e.g., J:208092 has 6 on both systems; J:206084 has 7 in P-WI but 11 on Public).
  • Expression Assays: includes all GXD expression assays and Cre activity assays associated with the reference.
Reference Summary for MGI Object
  • For a marker, the supported URL is /pwi/summary/reference?marker_id=MGI:(number)
  • For an allele, the supported URL is /pwi/summary/reference?allele_id=MGI:(number)
