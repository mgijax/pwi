P-WI Sequence Summary Functionality:

  • Displays sequence records associated with any Marker record.
  • Supported URL for a marker is /pwi/summary/sequence?marker_id=MGI:(number)
  • Description field is displayed verbatim, because angle brackets can be used.
  • Marker Symbol column displays multiple markers if there is more than 1 association.
  • Sequence ID links to external provider if possible.
  • The summary returns a maximum of 5,000 rows. If additional rows exist, display them by clicking the "View all..." link above the table, or downloading the tab-delimited file.
