P-WI Simple Vocabulary Editor Interface functionality

General Description

The Simple Vocabulary module is a display and edit page all in one. A search returns all vocabularies. You may not create a vocabulary from this module. You may only add/edit/delete terms. A user must be logged in to edit but can search without logging in.

  • Green = edit
  • Blue = read only
  • Yellow = current row

Supported Browsers

  • Chrome and FireFox are supported on both PC and Mac computers.
  • Safari is NOT supported for the PWI. There are features that do not work.

Searching

There is no searching by Vocabulary/Term attributes. The Search button simply returns all vocabularies.

Wildcards

  • No searching, no wildcards
