P-WI Specimen Summary Functionality:

  • Includes all types of specimens, both GXD and Cre. Cre specimens appear at the end of the table.
  • There is no row limit.
  • All specimens in all in situ type assays for a particular reference are shown (Blots are not in this summary).
  • Summary may also be downloaded as a tab-delimited file.
  • Sort order is:
    1. specimen label(ascii sort, so 10 comes before 1)
    2. gene
    3. MGI id of assay
  • ID of assay is a link to the assaydetail page
  • Notes:
    1. Age note - * indicates presence of age note, blank if field is null
    2. Specimen note (fully displayed), blank if field is null
