Vocabulary Term Detail Functionality:

URLs supported include:
  • /pwi/detail/vocterm/key/(number)
  • /pwi/detail/vocterm/MP:(number)
  • /pwi/detail/vocterm/GO:(number)
  • /pwi/detail/vocterm/EMAPA:(number)
  • /pwi/detail/vocterm/EMAPS:(number)
  • /pwi/detail/vocterm/OMIM:(number)
  • /pwi/detail/vocterm/DO:(number)
Displays the following data for any term.
  • Term
  • Vocabulary
  • Synonyms
  • Primary ID
  • Secondary (alternate) IDs, if any
  • Definition
  • Comment (public)
  • Number of paths to term
  • Term in DAG context with:
    • relationship type (part-of, is-a, negatively_regulates, etc.)
    • ancestors leading up to root term for the vocabulary
    • siblings
    • immediate child terms
